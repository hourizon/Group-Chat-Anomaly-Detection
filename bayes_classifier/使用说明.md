# 群聊异常行为检测器使用说明

## 快速开始

### 1. 测试运行

无需配置，直接运行测试：
```powershell
cd d:\Users\Lenovo\Desktop\d2l\v3.0-面试定制版方案\bayes_classifier
.\bot_main.exe
```

这将模拟检测流程，展示：
- 垃圾消息检测（spam probability > 0.95 → 删除）
- 快速加群检测（10分钟加群 ≥ 5 次 → 踢出）

---

### 2. 重新训练模型（可选）

```powershell
.\bayes_classifier.exe
```

这会：
1. 读取 `data/spam.txt` 和 `data/ham.txt` 训练语料
2. 生成新模型 `data/bayes_model.dat`

---

### 3. 自定义训练语料

编辑文本文件添加更多样本：

**data/spam.txt** - 垃圾消息样本（每行一条）：
```
buy now limited offer free money
click here for free prize
cheap pills online pharmacy
...
```

**data/ham.txt** - 正常消息样本（每行一条）：
```
hello how are you
meeting scheduled for tomorrow
thanks for your help
...
```

## 连接真实 QQ 机器人（需要 go-cqhttp）

### 1. 下载 go-cqhttp

从 https://github.com/Mrs4s/go-cqhttp/releases 下载 Windows 版本

### 2. 配置 go-cqhttp

编辑 `config.yml`：
```yaml
account:
  uin: 你的QQ号
  password: "你的密码"

# HTTP 服务
http:
  host: 127.0.0.1
  port: 5700

# 正向 WebSocket
ws:
  host: 127.0.0.1
  port: 6700
```

### 3. 启动 go-cqhttp

```powershell
./go-cqhttp_windows_amd64.exe
```

首次登录需要扫码认证

### 4. 启动检测器（需要 libcurl）

检测器通过 HTTP API 与 go-cqhttp 通信，需要 libcurl 库。简化版可直接运行 `bot_main.exe` 进行模拟测试。

---

## 编译项目

```powershell
# 编译完整检测引擎
g++ -std=c++17 -Wall -Iinclude -o bot_main.exe src/bot_main.cpp src/bayes.cpp src/profiler.cpp
```

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `bayes_classifier.exe` | 训练和测试分类器 |
| `test_profiler.exe` | 测试行为画像分析器 |
| `bot_main.exe` | 完整检测引擎（模拟测试） |

---

## 检测阈值

可在代码中修改：
- 垃圾消息阈值：`spam_prob > 0.95`（第70行）
- 快速加群阈值：`5 次 / 10 分钟`（第155行）
