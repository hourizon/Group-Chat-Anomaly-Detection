## Phase 1: 朴素贝叶斯分类器实现

### 2026-04-15 14:00 - 14:45

### 任务
- 设计用于存储词频和概率的数据结构
- 实现 Trainer 类，完成从文本文件学习并生成模型文件
- 实现 Classifier 类，完成加载模型并预测新文本概率

### 实现细节

1. **数据结构** (`bayes.hpp`)
   - `BayesModel`: 存储词频统计、先验概率
   - `Trainer`: 训练模块，从语料库学习
   - `Classifier`: 分类模块，应用贝叶斯公式

2. **核心算法**
   - 分词：基于空格和标点的简单分词
   - 概率计算：P(垃圾|词语) = 词语在垃圾中出现次数 / 垃圾总词数
   - 平滑处理：拉普拉斯平滑 (smoothing = 1.0)
   - 分类：log 概率求和避免下溢

3. **模型序列化**
   - 二进制格式保存模型到 `bayes_model.dat`

### 遇到的问题
- 编译错误：`std::log`, `std::exp` 未找到
  - 解决：添加 `#include <cmath>`
- 私有成员访问问题
  - 解决：添加 friend class 声明和 getWordCounts() 访问器

### 测试结果
```

buy now cheap pills online -> Spam: 0.9766 [SPAM]
hello how are you -> Spam: 0.0311 [HAM]
meeting tomorrow at 3pm -> Spam: 0.0141 [HAM]
```

## Phase 2: 统计画像分析器实现

### 2026-04-15 14:50 - 15:20

### 任务
- 实现 BehaviorProfiler 类，用 std::map 和 std::deque 维护用户行为时间序列
- 实现滑动窗口清理机制
- 实现异常判断逻辑

### 实现细节

1. **BehaviorProfiler**
   - 内存中维护用户行为队列
   - 滑动窗口：1小时（可配置）
   - 自动清理过期事件

2. **BaselineAnalyzer**
   - 计算观测值的均值和标准差
   - 支持 3σ 异常检测

3. **异常检测方法**
   - 绝对阈值：10分钟内加群 >= 5 次
   - 统计异常：超过均值 + 3σ

### 遇到的问题
- `getCurrentTimestamp()` 函数重复定义
  - 解决：删除重复定义
- 默认参数缺失
  - 解决：在头文件中添加默认参数 `window_ms = -1`

### 测试结果
```
user001 join events (1hr window): 4
user001 join >= 3? YES
Mean: 3, StdDev: 1.41421
10 anomalous? YES
```

## 面试要点总结

### 技术亮点
1. **从零实现**：不依赖任何机器学习库
2. **C++ 端到端**：训练、序列化、分类全部 C++ 实现
3. **算法驱动**：朴素贝叶斯 + 统计异常检测

### 核心算法原理
- **朴素贝叶斯**：基于贝叶斯定理 P(垃圾|文档) ∝ P(文档|垃圾) × P(垃圾)
- **统计异常**：基于均值和标准差的 3σ 法则
- **滑动窗口**：基于时间戳的实时行为分析

### 可扩展方向
1. 增加更多训练语料提升准确率
2. 支持多词组特征（bigram）
3. 实现 LDA 主题模型
4. WebSocket 实时连接
